收寄原则：

一、禁止收寄：禁止收取易燃、易爆、易腐蚀、有毒、放射性及违法物品。

二、限制收寄：

      1. 尺寸和重量：

        ①陆运快件：单件计费重量不超过130KG或三边（长、宽、高）分别不超过2.5米、1.5米、1.5米；

        ②航空快件：单件计费重量不超过80KG或三边（长、宽、高）分别不超过2.5米、1米、0.8米；

        特殊说明：顺丰时效产品需按照各时效产品的重量及尺寸要求执行。

      2. 航空限制：原商业包装液体、膏状、粉末类食品可通过航空方式发运，如原包装的奶粉、咖啡粉、保健品、矿泉水等；非商业原包装的其它液体、膏状、粉末类物品需提供《鉴定报告书》在发运机场报备通过后方可发运航空。
      3. 化工类：除日用化工品外（如牙膏、洗面奶、洗洁精、化妆品），其它化工类物品需提供安全属性证明材料，确认安全后，将证明材料随货寄出。若需要通过航空发运的，需提供《鉴定报告书》在发运机场报备通过后方可发运航空。
      4. 活体动物类：除大闸蟹、甲鱼、大龙虾外，其它活物体动物禁止收寄。
      5. 电池类：

        ①航空收寄标准：

            检测报告：必须提供符合UN38.3测试和1.2米跌落试验相关检查报告；

            报备：提供检测报告原件向机场安检报备通过后方可发运；

            包装：必须为商业包装，每台电子产品最多配2块电池，且机电分离，电池须独立密封包装；机电一体的电子产品须关机。

        ②电池禁收：禁止收取摩托车、汽车、电动车（含电动单车、三轮车、平衡车）的蓄电池，禁止收取额定能量大于100WH（瓦/小时）的锂电池或充电宝。

      6. 生鲜、时令水果类：为保证物品运输品质，部分时令蔬果类单票不能超过5KG，大闸蟹单票不能超过10KG，如超出需分票寄递，各地如有特殊要求，以当地收派员核实为准。

三、说明：

1、青海玉树、西藏地区暂只开通航空线路，快件需按照航空件的收寄标准执行。

2、在物品名称中未搜索到的物品原则上都可寄递，具体以收派员上门核实为准。
